# Setup

1. Create a public GitHub repository named exactly `nickihysterics`.
2. Put `README.md`, `assets/` and `.github/workflows/snake.yml` into that repository.
3. Push to the `main` branch.
4. Open the repository's **Actions** tab and run **Generate contribution snake** once.
5. Wait until the `output` branch appears. The snake image in the README will start rendering after that.

GitHub shows a profile README only when the repository name matches the account name.
